A personalized, immersive digital experience built with love.

This project is a modern web application designed to showcase memories, poetry, and heartfelt messages in a visually stunning, spatial interface inspired by the Apple Vision Pro.

## ‚ú® Features

- **Spatial Gallery (Vision Pro Style):**
  - A fully custom 3D carousel with glassmorphism effects, floating cards, and depth.
  - Features seamless auto-playing videos and high-quality photo integration.
  - Immersive "Spatial" UI with browser-style controls and artist attribution.

- **The Letter:**
  - A responsive, vertically scrolling love letter that serves as the heart of the experience.
  - Smooth reveal animations and swipe gestures for intuitive navigation.

- **Whispered Words (Poetry Book):**
  - An interactive, flip-book style poetry section.
  - Elegant typography and layout designed for reading comfort on any device.
